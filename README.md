# 🎙️ VoiceVerse Pro

**Conversational Voice Intelligence** – Speak and AI speaks back!

---

## ✨ Features

- 🎤 **Voice Input** – Speak naturally, AI transcribes using AssemblyAI
- 🧠 **AI Response** – Powered by Mistral AI with conversation memory
- 🔊 **Voice Output** – AI responds in voice using gTTS
- 🌐 **Multi-Language** – Auto-detect + 10 languages (Gujarati, Hindi, Portuguese, etc.)
- 📜 **Conversation History** – Remembers last 50 messages
- 📥 **Export Chat** – Download entire conversation as `.txt`
- 🎨 **Premium UI** – Dark glassmorphism design with typing indicator

---

## 🛠️ Tech Stack

| Component | Technology |
|-----------|------------|
| **Frontend** | Streamlit |
| **Speech-to-Text** | AssemblyAI |
| **AI Model** | Mistral AI |
| **Text-to-Speech** | gTTS (Google TTS) |
| **Language** | Python 3.x |
